1. Введение

Seedance 2.0 Prompt Studio — это локальное web-приложение для создания качественных промтов к моделям генерации видео (Seedance 2.0, Runway, Kling, Luma и др.). Приложение работает целиком в браузере, без сервера, и хранит все данные локально.

Что умеет

  • Собирать сложные кинематографичные промты по структурированным полям (subject, action, scene, camera, lighting, mood и др.)
  • Генерировать промт по короткой идее одним кликом через AI
  • Распознавать содержимое референсных картинок (Image-to-Video) с помощью AI Vision
  • Переводить с любого языка на кинематографичный английский
  • Создавать многосценные сценарии (Story Mode) с консистентным героем
  • Проверять сценарии на разрывы (Continuity Checker)
  • Экспортировать таймлайн в DaVinci Resolve / Final Cut Pro / Premiere через FCPXML
  • Сохранять историю, избранное, банк собственных героев и сцен

Что нужно

  • Современный браузер: Chrome, Edge, Firefox, Safari (последние 2 версии)
  • Файлы приложения: index.html и app.js в одной папке
  • Опционально: API-ключ OpenAI (или совместимого провайдера) для AI-функций
💡 Совет AI-функции опциональны. Без ключа приложение работает как ручной конструктор промтов — все базовые возможности доступны.

2. Запуск приложения

Локально

  1. Положи index.html и app.js в одну папку (например C:\Users\имя\Desktop\PROMT\).
  2. Дважды кликни на index.html — откроется в браузере по умолчанию.
  3. (Альтернативно) Открой через File → Open в браузере или перетащи в окно.
⚠ Важно Не переименовывай app.js — index.html ссылается на него по имени. Они должны лежать в одной папке.

На GitHub Pages / любом веб-сервере

Файлы статические, никаких сборок не требуется. Заливай оба файла в репозиторий → включай Pages → готова публичная ссылка.

3. Быстрый старт за 3 минуты

  1. Открой index.html в браузере.
  2. В поле «🎭 Subject (главный объект)» впиши: young woman with red hair
  3. В поле «🎬 Action»: walking through neon-lit city at night
  4. В поле «🌆 Scene»: cyberpunk Tokyo street, rainy reflections
  5. В выпадающем «🎨 Style» выбери cyberpunk.
  6. В выпадающем «💡 Lighting» выбери neon backlighting.
  7. Внизу справа в блоке «📜 Результат» нажми «📋 Скопировать (EN)».
  8. Готово — английский промт в буфере, вставляй в Seedance/Runway/Kling.
⚡ Ещё быстрее (1 клик с AI) Если настроил AI-ключ (раздел 5):
  1. В поле сверху «🎯 Цель» впиши на любом языке: реклама часов в стиле блокбастера
  2. Нажми 🎯 Goal mode.
  3. AI заполнит все поля английскими значениями. Жми 📋 Скопировать (EN).

3.5 🟢 Simple Mode (для блогеров и SMM)

Если ты не киномонтажёр и хочешь просто получить промт по короткой идее за 30 секунд — переключись в Simple Mode кнопкой в правом верхнем углу (🟢 Simple ↔ 🎛 Pro).

💡 Кому подходит Simple Mode SMM-специалисты, контент-менеджеры, блогеры, маркетологи, копирайтеры — все, кому нужен готовый промт для Sora/Runway/Kling/DALL·E, а не тонкая настройка 30 параметров. Опытные пользователи всегда могут перейти в Pro mode для дошлифовки.

3.5.1 📝 Text Mode (текст → профессиональный промт)

Самая мощная вкладка. AI извлекает «ядро идеи» (subject + action + object), фиксирует его, и строит вокруг него 10-блочный кинематографичный промт.

Шаги

  1. Открой Simple Mode → вкладка 📝 Текст.
  2. Введи идею на любом языке: мальчик смотрит мультик
  3. Выбери цель: 🎬 Видео или 🖼 Картинка.
  4. Выбери aspect ratio: 16:9, 9:16, 1:1, 4:3 и т.д.
  5. Выбери стиль: 🎬 Cinematic, 📱 TikTok, 🎨 Anime, 🧸 Pixar, 🌫 Dreamy и др.
  6. Выбери длину: Compact (50w), Standard (100w), Detailed (180w).
  7. Жми ✨ Создать.

Что появится на выходе

  • 🔒 Зафиксированное ядро — фиолетовая плашка показывает извлечённый subject + action + object. Это гарантия, что AI не уйдёт в сторону от твоей идеи.
  • 3 варианта с разной подачей: action-led / emotion-led / visual-led.
  • На каждом — score 0-100 (потенциал вовлечённости) и короткое обоснование.
  • Если AI потерял ядро — появится красное предупреждение и кнопка 🔒 Восстановить ядро.
📐 10-блочная структура промта Каждый промт строится по проверенной киношной схеме: subject · action · environment · camera · lens · lighting · palette · mood · style · technical. Это формат, который любят все современные видео-модели.

3.5.2 🖼 Image Mode (текст → картинка через DALL·E 3)

Вкладка для быстрой генерации картинок без сложных настроек.

Шаги

  1. Вкладка 🖼 Картинка.
  2. Опиши картинку на любом языке: уютная кофейня в Токио в дождь
  3. Выбери размер: 1024×1024 (1:1), 1792×1024 (16:9), 1024×1792 (9:16).
  4. Выбери стиль: Cinematic, Photorealistic, Anime, 🧸 Pixar, Watercolor, Cyberpunk, Minimal, Vintage и др.
  5. Жми 🖼 Сгенерировать.
🌐 Авто-перевод RU → EN Если ты ввёл идею по-русски, приложение сначала прогонит её через AI-переводчик в концентрированный английский (DALL·E работает с английским гораздо лучше), и только потом отправит в модель. На выходе ты увидишь оба варианта — что было отправлено и твой исходник.

Стили (10 шт)

🎬 Cinematic · 📷 Photorealistic · 🎨 Anime · 🧸 Pixar · 🖌 Watercolor · 🌃 Cyberpunk · ✏ Minimal · 📼 Vintage · 🎭 Oil Painting · 🌸 Pastel

Стиль добавляет специально оптимизированный suffix к промту. Pixar, например: "3D animation, expressive cartoon character, oversized eyes, soft subsurface scattering, octane render".

3.5.3 🎬 Video Mode (3 варианта по шаблону)

Вкладка для генерации 3 разных видео-промтов по выбранному шаблону формата.

Шаги

  1. Вкладка 🎬 Видео.
  2. Выбери шаблон формата (плитки): TikTok hook, Reels story, UGC product, Cinematic ad, Tutorial и др. Каждый шаблон уже содержит правильный aspect, длительность, стиль съёмки.
  3. Опиши идею: премиум-кофе утром на даче
  4. Опционально: Brand (название продукта/бренда) и Message (эмоциональный посыл).
  5. Жми ✨ Создать.

На выходе — 3 принципиально разных промта:

  • v1: action-led — упор на динамику, движение
  • v2: emotion-led — упор на эмоции, лица, чувства
  • v3: product/visual-led — упор на продукт и визуальную эстетику

Все 3 со score виральности 0-100 (отсортированы) и обоснованием на русском.

3.5.4 🌐 EN + RU — двуязычный вывод

С версии 10.4 каждый сгенерированный промт показывается на двух языках одновременно:

ЯзыкЗачемГде видно
🇬🇧 ENДля вставки в AI-модели (Sora, Runway, Kling, Veo, DALL·E). Английский — родной язык этих моделей, точность результата выше на 30-50%.Развёрнут по умолчанию, в выделенной рамке.
🇷🇺 RUДля понимания, что AI заложил в промт. Помогает увидеть, не потерялось ли ядро идеи. Удобно для отчётов и презентаций клиенту.Свёрнут под кнопкой ▼ RU — перевод для понимания.
🔄 Перевод, не пересказ RU-версия — точный перевод того же промта (тот же subject, action, те же блоки camera/lens/lighting), а не отдельный промт «по мотивам». Технические термины (anamorphic, dolly in, 24fps) сохраняются латиницей или в скобках.

Кнопки копирования

  • 📋 EN — копирует английский (используй для AI-моделей)
  • 📋 RU — копирует русский (используй для отчётов)

Все интеграции (🖼 → В картинку, 🖼 Превью, 🎬 → В Pro mode) автоматически передают английскую версию, потому что DALL·E и Pro mode тоже лучше работают с английским.

5.3 Безопасность ключа

Ключ не передаётся никуда кроме указанного API. Хранение:

  • Ключ обфусцируется через XOR + base64 и сохраняется в localStorage с префиксом XOR:
  • На каждом AI-запросе расшифровывается на лету
  • Никогда не появляется в логах в plaintext
⚠ Это обфускация, не криптография Защита от случайного просмотра localStorage. Если кто-то может выполнить JS на твоём компьютере — ключ всё равно вытащат. Не используй приложение на чужой машине.

7. Image-to-Video режим

Если у тебя есть референсная картинка и ты хочешь чтобы видео началось с неё — переключись в режим I2V.

Шаги

  1. Внизу формы переключи режим: T2V → I2V.
  2. Перетащи картинку в зону «🖼 Первый кадр» (или нажми «выбрать файл»).
  3. Если AI-ключ настроен — Vision автоматически распознает картинку и заполнит поля subject/scene/style/lighting и др.
  4. (Опционально) Загрузи второе изображение в «🖼 Последний кадр» — если модель поддерживает start+end frames.
  5. Жми 📋 Скопировать (EN).
💡 Vision запрашивает подтверждение При первой загрузке появится диалог «Распознать через AI Vision?». Жми OK — больше за сессию не спросит.
⚠ Vision требует gpt-4o (не mini) gpt-4o-mini технически имеет vision, но качество хуже. Для I2V переключайся на gpt-4o в настройках AI.

8. Multi-shot (раскадровка)

Когда нужно несколько связанных кадров в одном видео (например, продукт показан с 3 ракурсов):

Активация

  1. Найди переключатель «🎞 Multi-shot» и включи его.
  2. Появится таблица шотов с кнопкой «➕ Добавить шот».

Поля шота

ПолеЧто вписать
Длительность3s / 5s / 8s
Cameramedium shot, slow dolly in
ActionЧто происходит в этом шоте
Transitioncut / dissolve / match cut

Шоты можно перетаскивать (drag handle ⠿) для смены порядка. Character в основной форме обеспечивает консистентность.

9. Timeline beats

Если нужна чёткая хронология внутри одного шота (что происходит на 0с, 2с, 4с):

  1. Включи «⏱ Timeline beats».
  2. Добавь точки времени с описанием камеры и subject в каждый момент.
  3. Это попадёт в промт как at 0s: ... at 2s: ...

10. Story Mode (multi-scene)

Самая мощная фича — превращает одну идею в N связанных сцен с консистентным героем.

Открыть

Кнопка 🎬 Story в шапке справа, или Command Palette → Story Mode.

Шаги

  1. В поле «💡 Большая идея» впиши концепт на любом языке: детектив расследует исчезновения в туманном порту
  2. Выбери жанр/тон: Neo-noir, Cyberpunk thriller, Anime adventure и др. (11 опций)
  3. Слайдером «🎚 Кол-во сцен» выбери от 3 до 10
  4. Выбери длительность сцены: 3s / 5s / 8s / 10s

Результат

  • Hero — общее описание героя (3–5 визуальных черт), консистентное во всех сценах
  • Title — короткое название истории
  • Scenes[] — N карточек, каждая с заголовком и полным cinematic-английским промтом

11. Continuity Checker

AI-скрипт-супервайзер находит разрывы между сценами и предлагает фиксы.

Запуск

В Story Mode → кнопка 🔍 Continuity check. AI читает героя + все сцены, отвечает за 5–15 секунд.

Что проверяет (8 типов)

ТипПример разрыва
👔 wardrobeScene 1: куртка → Scene 2: смокинг (без причины)
⏰ timeScene 2: ночь → Scene 3: день (без перехода)
☔ weatherДождь → солнечно мгновенно
🎒 propСумка появилась/исчезла
📍 locationГород → лес → город
💡 lightingКонтражур → плоский свет в той же локации
🧬 characterЦвет волос/глаз героя поменялся
❓ otherЛюбые другие несоответствия

Что показывает

  • Score 0–100 — общая оценка сценария (зелёный ≥80, жёлтый ≥50, красный <50)
  • Issues — список разрывов с severity 🔴 high / 🟡 med / 🔵 low
  • Для каждого: какие сцены затронуты, описание на русском, готовый английский fix-текст

Действия

  • 🔧 Применить fix — AI переписывает целевую сцену устраняя разрыв

12. Экспорт FCPXML

Превращает Story Mode в готовый таймлайн для монтажки.

Что в архиве

story-fcpxml-1730000000.zip
├── story.fcpxml      ← таймлайн (FCPXML 1.10)
├── media/
│   ├── preview-01.png   (real или placeholder)
│   ├── preview-02.png
│   └── ...
├── story.json        (резерв-копия сценария)
└── README.txt        (инструкция импорта)

Что в .fcpxml

  • Sequence 1920×1080 @ 24fps
  • Spine — N клипов подряд, длина = выбранный storyDur
  • Имена клипов = названия сцен («Scene 1: Hero arrives»)
  • <note> на каждом клипе = полный английский промт
  • <marker> на клипах = continuity issues (если делал проверку)

13. Импорт в DaVinci Resolve

Подготовка

  1. Распакуй ZIP. Не оставляй упакованным — Resolve должен видеть media/ рядом.
  2. Положи в удобное место: Documents/Seedance/My Story/

Импорт

  1. Открой DaVinci Resolve → New Project → дай имя.
  2. Меню: File → Import → Timeline...
  3. Выбери story.fcpxml
  4. В диалоге «Load XML»: укажи Source clips path = папка с распакованным ZIP
  5. Галочки оставь по умолчанию → OK

После импорта

  • В Media Pool появятся PNG-превью + объект таймлайна с именем твоей истории
  • Двойной клик по таймлайну → открывается на дорожке
  • Видно N клипов подряд, подписанных, с маркерами continuity

Замена placeholder на реальное видео

  1. Сгенерь mp4 через Seedance/Runway/Kling (используя промт из <note>)
  2. Перетащи mp4 в Media Pool
  3. Зажми Alt (Win) / Option (Mac) и перетащи новый клип на placeholder — Resolve заменит сохранив длину/позицию
  4. Альтернатива: правый клик на клипе → Replace Clip

Final Cut Pro

Меню: File → Import → XML... → выбери story.fcpxml

Premiere Pro

⚠ Premiere ограничен FCPXML 1.8 Workaround: импортируй сначала в Resolve → File → Export → Final Cut Pro 7 XML → этот legacy-файл откроется в Premiere.

22. Решение проблем

AI-кнопки нажимаются, но ничего не происходит
1) Проверь что в ⚙ AI вписан ключ. 2) Открой DevTools (F12) → Console → ищи красные ошибки про 401/403/CORS. 3) Проверь что хватает $ на балансе у провайдера.
Vision не распознаёт картинку в I2V
Vision требует gpt-4o или gpt-4-vision-preview. Зайди в ⚙ AI → смени модель. gpt-4o-mini работает плохо для Vision.
Story Mode возвращает «AI вернул не сценарий»
Локальные модели (Ollama) часто не держат JSON-формат. Используй gpt-4o или claude-sonnet. Слабые модели (mini) тоже могут портить JSON — переключись на старшие.
localStorage заполнился, ошибка quota
Превью-картинки в Story Mode тяжёлые. Очисти историю/избранное (🗑) или удали старые сценарии. Размер квоты браузера — обычно 5–10 MB.
FCPXML импортируется в DaVinci, но клипы красные («Media offline»)
Resolve не нашёл папку media/. Правый клик в Media Pool → Relink Selected Clips → укажи папку с распакованным ZIP.
Premiere ругается на FCPXML 1.10
Premiere поддерживает только до 1.8. Импортируй в Resolve → File → Export → Final Cut Pro 7 XML → этот файл откроется в Premiere.
Голосовой ввод не работает
В Firefox/Safari Web Speech API ограничен. Используй Chrome или Edge. Также нужен HTTPS или localhost — на file:// может не работать.
Поля заполняются «[object Object]»
Старая версия. Убедись что у тебя app.js v7+. Это было исправлено через функцию applyAiFields.
Кнопка ★ Избранное не работает
Сначала нужно сгенерировать промт (нажать любую кнопку чтобы заполнился блок результата). Без английского промта избранное пустое.
Хочу сбросить всё к заводским настройкам
F12 → Console → введи localStorage.clear() → Enter → перезагрузи страницу. Сотрутся все настройки, история, ключ, темы, story.

23. FAQ

Сколько стоит использовать?
Само приложение бесплатно. AI-функции — по тарифам твоего провайдера. С gpt-4o-mini один промт ≈ $0.0001, story из 5 сцен ≈ $0.001. Превью DALL·E ≈ $0.04 за кадр. Локальный Ollama — $0.
Можно ли работать без интернета?
Да. Без AI — полностью офлайн. С Ollama (локальная модель) — AI тоже офлайн.
Где хранятся мои данные?
Только в localStorage твоего браузера. Никаких облаков, никакой телеметрии. Очистка кэша браузера = потеря данных.
Можно поделиться сценарием с коллегой?
В Story Mode → 📦 Экспорт ZIP или 📄 Экспорт .txt. Коллеге пришли файл — он откроет.
Поддерживается ли мобильный?
Интерфейс адаптивный, но лучше работать с десктопа. Голосовой ввод и drag-drop для I2V на мобильных ограничены.
Можно сделать публичную ссылку?
Да — залить два файла на GitHub Pages, Netlify, Vercel. Папка статическая, сборки не нужно.
Где брать API-ключи?
OpenAI: platform.openai.com/api-keys. Anthropic: console.anthropic.com. OpenRouter: openrouter.ai/keys. Ollama: установи с ollama.com, ключ не нужен.
Какая модель Seedance/Runway лучше?
Промт-структура одинаковая для всех. Skip-test: возьми один и тот же промт из приложения и прогони через несколько генераторов — выбирай по результату.
